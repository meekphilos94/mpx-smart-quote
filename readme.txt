Professional AJAX-driven shipping cost calculator for MPX Global, featuring PDF quotes, multi-origin rates, and WhatsApp integration.

== Description ==

The **MPX Global Shipping Calculator** is a robust, user-friendly WordPress plugin designed specifically for logistics and shipping companies. It allows customers to get instant, accurate shipping estimates for importing goods from varying origins (China, Dubai, USA, UK, India, Turkey) to Zimbabwe.

**Key Features:**

*   **Multi-Origin Support**: Distinct rates and modes for China, Dubai, USA, UK, India, and Turkey.
*   **Dynamic Calculations**: Calculates cost based on weight (kg), quantity (pc), or volume (CBM) depending on the shipping mode.
*   **Volumetric Weight**: Automatically checks for volumetric weight if dimensions are provided.
*   **PDF Quotes**: Clients can download a professional PDF quote of their estimate.
*   **WhastsApp Integration**: Generate a pre-filled WhatsApp message to send the quote directly to the sales team.
*   **Admin Settings**: Easily manage shipping rates, transit times, addresses, and currency numbers from the WordPress dashboard.
*   **Responsive Design**: A sleek, modern UI that works perfectly on desktop and mobile devices.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/mpx-smart-quote` directory, or install the plugin through the WordPress plugins screen.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to **MPX Calculator** in the admin menu to set up your shipping rates and addresses.
4.  Use the shortcode `[mpx_calculator]` on any page to display the calculator.
